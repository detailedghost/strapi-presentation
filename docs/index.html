What is Strapi?

A headless Content Management System (CMS) built on NodeJS

strapi.io

It's a simple way for us to serve images, data, and even build out simplified microservices. We do this to make our devs lives easier when they build out a Single Page Application.

A BaaS!!

backend-as-a-service

Quick Overview of Concepts

  • Collections Types > a set of things we want to serve to our web app
  • Single Types > a single object
  • Plugins > community built addons
  • Permissions > what users can do
  • Roles > a set of permissions a user can have

Demo with Concepts

The Good

  • Quick Setup, and easy to self host
  • Easy to use interface
  • Light, and fast
  • Microservice friendly
  • Extensions! Plenty of Plugins
  • Customizable to the businesses needs

The Bad

  • No database migration support (yet)
  • Media management dashboard is basic
  • Too flexible for non-developers
  • No true multi-site support
  • Internationalization Features are hit and miss

My Suggestions

It's great for building simple sites, with targeted purposes. If it needs to be extended, tread lightly
